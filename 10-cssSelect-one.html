1、属性选择器

  • 新增三个属性选择器
  • E[attr^=val],匹配元素E，且E中包含了 以属性值val开头的任何字符串： a[class^=column]{ }
  • E[attr$=val],匹配元素E，且E中包含了 以属性值val结尾的任何字符串： a[href$=doc] { }
  • E[attr*=val],匹配元素E，且E中定义了属性值val任何字符串，类似模糊查询： a[href*=doc] { }
  • 可以参考 正则匹配表达式符号记忆
